Chrome插件

发表于 2019-09-30 | 分类于 杂货铺 | |

中午整理了一下Chrome的插件，想了想把各个插件大体作用写一下吧。本来早该写完，下午被隔壁拉去打csgo了，并且白给了一下午

https://chrome.google.com/webstore/category/extensions?utm_source=InfinityNewtab

Adblock

拦截网页上的广告。

infinty

新标签页,实现收藏网页手机桌面效果


Chrono下载管理器

Chrome浏览器中下载管理器

Nimbus 截幕 & 屏幕录像机

全屏幕捕获网页或任何部分。编辑的截图。记录的视频记录的视频从您的屏幕

Wappalyzer

它可以检测 内容管理系统， 电子商务平台， Web框架， 服务器软件， 分析工具和 更多。

Proxy SwitchyOmega

轻松快捷地管理和切换多个代理设置。

IP Whois & Flags Chrome & Websites Rating

Display Website Country Flag. Website Whois - how many Visitors are visiting, Reputation Reviews, Antivirus Check, Alexa, Incognito

Shodan

告诉您网站的托管位置（国家，城市）开放端口

ModHeader

修改请求和响应头

User-Agent Switcher for Chrome

模仿访问方式 win iOS mac 安卓

EditThisCookie

EditThisCookie是一个cookie管理器。您可以添加，删除，编辑，搜索，锁定和屏蔽cookies！

IP Address and Domain Information

最终在线调查工具 ！请参阅每个 IP 地址、 域名和提供程序的详细的信息。

My IP address

找到您的 IP 地址 ！请参阅 ISP、 位置和 DNS。获取 IP 地址更改通知，看到您的历史记录。

X-Forwarded-For Header

更换访问ip

Charset

修改网站的默认编码

Whois Website Hosting Company Info & IP Whois

Whois tool to lookup any Website Live Hosting Company, IP, Websites on this IP Now/Before, Site IP Change history, Nameservers, etc

SimilarWeb - 网站流量来源和排名

通过SimilarWeb扩展程序查看真实的深入网站参与、流量来源和网站排名信息。

类似的网站 - 发现相关网站

立即发现与您当前正在浏览的网站类似的网站。探索网络，了解更多信息。

JSON Viewer Awesome

该扩展程序可帮助您在浏览器中可视化来自任何网站或API请求的JSON响应。它为您提供了很棒的JSON美化体验。它具有许多独特的功能，使其超赞。它可以帮助开发人员格式化或修饰JSON并遍历图形视图中的属性

ApiDebug-Http接口调试插件-中文版

ApiDebug 可用于http、https等接口调试。是一个开源API接口调试插件，支持Restfull接口调试、POST请求模拟，同时也支持PUT、DELETE等接口

Web Developer

添加带有各种web开发工具的工具栏按钮。

XPath Helper

轻松地提取、编辑和计算xpath查询。 按住ctrl + shift

https://jingyan.baidu.com/article/1e5468f94694ac484861b77d.html

WebRTC Network Limiter

隐藏用户上网的真实IP地址 Chrome插件

d3coder （decoder，解码器）

这项扩展将添加一个右键菜单，方便渗透测试人员按照指定格式编码/解码文本

这就是我比较常用的一些插件，如有错误请表哥指正
